# Car Inspection App

A buildless React application for vehicle inspections, featuring digital forms, visual damage mapping, and PDF report generation.

## Features

- **Vehicle Information**: Digital entry for VIN, Make, Model, etc.
- **Condition Checklist**: Interactive Pass/Fail checks.
- **Visual Damage Inspector**: Interactive schematic for marking car damage.
- **PDF Export**: Generate professional reports with images.
- **Offline Storage**: Uses browser LocalStorage for data persistence.

## How to use locally

1. Clone this repository.
2. Open `index.html` in your browser.
3. No build steps are required as it uses CDN-loaded React and Babel.

## Deployment

This project is ready to be deployed on **Vercel** or **GitHub Pages**. 
Simply connect your repository to Vercel and it will detect it as a static project.
